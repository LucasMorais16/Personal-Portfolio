Memory Match Card Game

About

Developed during my first year at university, this project began as a console-based memory match game for the Windows terminal. However, two classmates and I challenged ourselves to create a fully graphical interface version.

The game features multiple difficulty levels: 'Easy' and 'Normal' differ in both memorization time and scoring per match. The 'Mãe Dinah' level—named after a famous Brazilian fortune teller—adds an extra challenge, allowing players to guess card positions with higher points awarded for correct matches.

A leaderboard system—accessible through the menu—displays the top 10 players' scores.

Additionally, a comprehensive save system stores both individual gameplay progress and the leaderboard data.

Tools

  • Visual Studio 2019/2022
